
This is a package for audio recording and encoding on Android.



The recording code is based on the AudioBoo android app:
https://github.com/Audioboo/audioboo-android

For license and more informationsee
https://github.com/Audioboo/audioboo-android/blob/master/LICENSE



All encoding is done with libsndfile, sources are downloadable at
http://www.mega-nerd.com/libsndfile/

For license and more information see
sndfile/README.txt
