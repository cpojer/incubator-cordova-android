Compiling for Android
---------------------

See the following blogpost:
http://rodolfotech.blogspot.co.at/2011/01/compiling-libsndfile-for-android-ndk-r5.html
and github repository:
https://github.com/michaelwu/libsndfile

They explain how to compile libsndfile for Android.

I copied all files from libsndfile/src to this project and then added them
to Android.mk and Application.mk.
The files config.h and sndfile.h, which are generated by the configure script,
are also added to libsndfile/src.

Have Fun!
